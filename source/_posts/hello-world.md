## Quick Start
<!-- more -->
### Create a new post

``` bash
$ hexo new "My New Post"
```
or
``` bash
$ hexo n "your name"
```

### Run server

``` bash
$ hexo server
```

### Generate static files

``` bash
$ hexo generate
```

### Deploy to remote sites

``` bash
$ hexo deploy
```
